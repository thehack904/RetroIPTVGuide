# AI Usage Policy

RetroIPTVGuide uses AI-assisted development tools as part of the development workflow.

AI is used to assist with:

- code suggestions
- refactoring ideas
- documentation drafting
- architecture exploration
- troubleshooting assistance

All AI-generated content is reviewed, modified, and validated by a human maintainer before being committed to the repository.

## Development Principles

RetroIPTVGuide is not developed through automated or unsupervised code generation.

All contributions must:

- be reviewed for correctness
- be validated through testing
- follow the project architecture
- maintain compatibility with existing systems

AI tools are treated as **development assistants**, similar to IDE autocomplete or static analysis tools.

## Security Considerations

AI-generated code is subject to the same review standards as manually written code.

Before merging changes:

- dependencies are reviewed
- potential security issues are evaluated
- functionality is verified against project requirements
